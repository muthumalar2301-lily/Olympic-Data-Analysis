# 🥇 Olympic Data Analysis

## 📌 Project Overview

This project analyzes over 120 years of Olympic history using Python and Exploratory Data Analysis (EDA). The objective is to discover meaningful insights about medal distribution, athlete performance, sports popularity, gender participation, and Olympic participation trends.

## 🎯 Objectives

* Analyze medals won by different countries
* Identify the most successful Olympic athletes
* Explore the popularity of Olympic sports
* Compare male and female athlete participation
* Study participation trends across Olympic years

## 🛠️ Technologies Used

* Python
* Pandas
* Matplotlib
* Seaborn
* Jupyter Notebook

## 📂 Dataset

* athlete_events.csv
* noc_regions.csv

Dataset: **120 Years of Olympic History: Athletes and Results**

## 📊 Analysis Performed

* Data Cleaning
* Top 10 Countries by Medal Count
* Top 10 Athletes by Medal Count
* Most Popular Olympic Sports
* Gender Participation Analysis
* Olympic Participation Trend
* Medal Distribution
* Country Participation Analysis
* Correlation Heatmap

## 📈 Key Insights

* The United States has won the highest number of Olympic medals.
* Athletics and Swimming are among the most popular Olympic sports.
* Male participation has historically been higher, while female participation has steadily increased.
* Olympic participation has grown significantly over the years.
* Height and weight show a positive correlation among athletes.
